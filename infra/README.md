# infra

Deployment and operator infrastructure templates live here.

## Reserved Areas

- `deployment/`: deployment topology notes and templates.
- `docker/`: Docker-related documentation only unless explicitly activated.
- `nginx/`: reverse-proxy templates.
- `postgres/`: PostgreSQL operator notes.
- `systemd/`: Linux service templates.

Do not start Docker, provision live infrastructure, or add production deployment
behavior during backend documentation/foundation milestones.
