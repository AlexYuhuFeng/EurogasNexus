# infra/systemd

Linux service notes live here.

## Status

Reserved for deployment documentation. Do not add service units until a release
or deployment milestone activates this directory.

## Future Scope

When activated, this directory may contain:

- API service unit templates;
- worker service unit templates;
- scheduler service unit templates;
- environment-file examples that contain variable names only;
- restart and logging policy notes.

## Secret Rule

Never commit real environment files, database URLs, API keys, tokens, or
operator credentials.
